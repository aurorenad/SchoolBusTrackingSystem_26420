# SchoolBusTrackingSystem_26420


A School Bus Tracking System designed to enhance student safety and provide real-time bus monitoring for parents, school administrators, and drivers.

##  Project Overview

This system enables schools to track buses in real-time, monitor student pickups and drop-offs, and communicate effectively with parents and administrators.

###  Objectives

* Improve safety and accountability
* Provide live GPS tracking of school buses
* Allow parents to know bus locations and arrival times
* Help administrators manage transportation efficiently

## Tech Stack

* **Java** (Core application logic)
* **Spring Boot** (Backend API)
* **PostgreSQL** (Database)
* **Hibernate/JPA** (ORM)
* **Maven** (Dependency Management)

## 📌 Features

✅ Driver & Bus Management
✅ Student and parent Registration

## 🧩 System Modules

| Module              | Description                                     |
| ------------------- | ----------------------------------------------- |
| User Module         | Admin, Parent and Teachers                      | 
| Bus Management      | Register and track buses                        |
| Student Management  | Assign students to buses, manage records        |
| Route Management    | Define and update bus routes                    |
| Driver Mangement    | record driver and assign them to a bus          |
| Location Management | record, update, delete location                 |


Main Entities include:

* User
* Bus
* Driver
* Student
* Route
* Location
